# MapVina Native for iOS

[![GitHub Action build status](https://github.io/github/mapvina/mapvina-native/actions/workflows/ios-ci.yml/badge.svg)](https://github.io/github/mapvina/mapvina-native/actions/workflows/ios-ci.yml) [![CocoaPods Package](https://img.shields.io/cocoapods/v/MapVina.svg)](https://github.com/CocoaPods/Specs/tree/master/Specs/b/0/0/MapVina) [![](https://img.shields.io/endpoint?url=https%3A%2F%2Fswiftpackageindex.com%2Fapi%2Fpackages%2Fmapvina%2Fmapvina-gl-native-distribution%2Fbadge%3Ftype%3Dplatforms)](https://swiftpackageindex.io/github/mapvina/mapvina-gl-native-distribution)
